## To Build

```bash

```

## To Run
```bash
java -cp ./out Main <db loc> <page size> <buffer size>
```

Arguments:
- **db loc**:         Path to the database root
- **page size**:      Size of page ( in bytes )
- **buffer size**:    Size of page buffer ( page capacity )
